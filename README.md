# kotlin-spring

## Setup
Copy the repository and run:
```bash
$ ./gradlew bootRun
```

Or download the JAR and run:
```bash
$ java -jar kotlin-rest-service-1.0.0.jar
```

The application run on `localhost:8080/`

Swagger docs are available at `localhost:8080/swagger-ui.html`
